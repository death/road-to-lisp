# Road to Lisp

The purpose of this repository is to archive submissions to the Road
to Lisp Survey.  The initial entries come from a Wayback Machine
[snapshot](https://web.archive.org/web/20120106121645/http://wiki.alu.org/The_Road_To_Lisp_Survey)
of the ALU wiki.  Tags were not included.

If you would like to submit your own entry, the [template
file](Template.md) is a good start.  Corrections, updates, and other
feedback from the persons involved is also welcome.

You may also be interesting in [Cliki's Person
page](https://www.cliki.net/Person).
